## Essential Daily Shortcuts
| Shortcut | Action |
|---|---|
| `Cmd+S` | Save — make this a reflex |
| `Cmd+Z` | Undo |
| `Cmd+Shift+Z` | Redo |
| `Cmd+J` | Toggle terminal panel |
| `Cmd+Shift+P` | Command palette — access any VS Code command |
| `Cmd+P` | Quick open / find file by name |

---

## Files and Editor
| Shortcut | Action |
|---|---|
| `Cmd+N` | New file |
| `Cmd+W` | Close current tab |
| `Cmd+Shift+N` | New window |
| `Cmd+Tab` | Cycle through open tabs (hold Cmd) |
| `Cmd+Shift+E` | Toggle file explorer sidebar |
| `Cmd+B` | Toggle sidebar visibility |

---

## Editing
| Shortcut | Action |
|---|---|
| `Cmd+/` | Toggle line comment |
| `Option+Up/Down` | Move current line up or down |
| `Cmd+D` | Select next occurrence of highlighted word |
| `Cmd+Shift+K` | Delete current line |
| `Option+Shift+Down` | Duplicate current line below |
| `Cmd+Enter` | Insert new line below (without breaking current line) |
| `Cmd+Shift+Enter` | Insert new line above |
| `Tab` | Indent line |
| `Shift+Tab` | Outdent line |

---

## Search and Replace
| Shortcut | Action |
|---|---|
| `Cmd+F` | Find in current file |
| `Cmd+H` | Find and replace in current file |
| `Cmd+Shift+F` | Find across all files in project |
| `Cmd+Shift+H` | Find and replace across all files |

---

## Terminal
| Shortcut | Action |
|---|---|
| `Cmd+J` | Toggle terminal panel open/closed |
| `Cmd+Shift+`` ` | Create a new terminal instance |
| `Cmd+C` | Cancel running command (inside terminal) |
| `clear` | Clear terminal screen |

## Navigation
| Shortcut | Action |
|---|---|

| `Cmd+G` | Go to line number |
| `Cmd+Shift+O` | Go to symbol in file |
| `Cmd+Left/Right` | Jump to start/end of line |
| `Cmd+Up/Down` | Jump to top/bottom of file |
| `Option+Left/Right` | Jump word by word |
